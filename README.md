## 📦 Getting Started
```
git clone https://github.com/garu2/TodoList-ReactJs-Supabase.git
```
```
cd TodoList-ReactJs-Supabase
```
```
npm install
```
```
npm run dev
```
The app will be available at `http://localhost:5173`
